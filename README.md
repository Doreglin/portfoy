# 🚀 Kişisel Portfolyo Web Sitesi | Umut Sırakaya

Bu proje, kişisel projelerimi, deneyimlerimi ve yazılım serüvenimi sergilediğim, modern bir arayüze sahip duyarlı (responsive) bir portfolyo web sitesidir. Tamamen HTML5 ve CSS3 kullanılarak geliştirilmiştir.

![Hakkımda Bölümü](Ekran%20görüntüsü%202026-03-11%20144249.png)

## ✨ Özellikler

- **Modern ve Şık Tasarım:** CSS değişkenleri (custom properties) kullanılarak oluşturulmuş tutarlı renk paleti, karanlık tema ve neon gölgelendirmeler.
- **Duyarlı (Responsive) Yapı:** Masaüstü, tablet ve mobil cihazlarla tam uyumlu görünüm.
- **Glassmorphism Efektleri:** Arka planı bulanıklaştırılmış, şeffaf ve sabit navigasyon çubuğu (header).
- **Projeler Vitrini:** Geliştirdiğim oyun ve web uygulamalarının detaylı sunumu.

## 💻 Sergilenen Projeler

Portfolyo üzerinde detaylarını görebileceğiniz başlıca projelerim:

![Projeler Bölümü](Ekran%20görüntüsü%202026-03-11%20144303.jpg)

1. **S@TRAT E-Ticaret Sitesi:** Standart e-ticaret mantığını yıkan, siberpunk ve RPG elementleriyle donatılmış Full-Stack web uygulaması.
2. **Karanlık Zindan: Ultimate Edition:** Tarayıcı üzerinde çalışan, derin RPG mekaniklerine, roguelite elementlerine ve köy kurma sistemine sahip karanlık fantezi sıra tabanlı oyun.
3. **FilmBul:** Vanilla JavaScript ile geliştirilmiş modern film arama ve izleme listesi oluşturma uygulaması.

## 🛠️ Kullanılan Teknolojiler

- **HTML5:** Semantik web yapısı
- **CSS3:** Flexbox, CSS Değişkenleri, Medya Sorguları (Media Queries) ve Animasyonlar
- **Fontlar & İkonlar:** Google Fonts (Poppins), Font Awesome, Bootstrap Icons
